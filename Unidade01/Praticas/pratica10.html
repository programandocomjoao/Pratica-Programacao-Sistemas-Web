Linguagem de Marcação (HTML)

Em princípio, para se criar uma página web, é necessário um editor de textos, que pode ser o próprio Notepad (Bloco de Notas) do Windows ou qualquer outro que esteja disponível, para realizar a digitação código-fonte da página, que é composto por texto e tags HTML, e um browser (navegador web) para testar e visualizar o resultado do que foi feito no código-fonte.

O HTML é a linguagem de marcação padrão para criação de páginas web. HTML significa Hyper Text Markup Language (Linguagem de Marcação de Hipertexto). É utilizado para descrever a estrutura de páginas web adicionando marcação significativa aos elementos da página. Os elementos HTML são os blocos de construção das páginas HTML.

As tags HTML são palavras reservadas que são interpretadas pelo navegador web para informar de que forma o texto que está sendo envolvido por elas devem ser exibidos ao usuário. Elas são identificadas no código-fonte por estarem entre os sinais de menor e maior.

A maioria dessas tags deve ser aberta antes do conteúdo que elas vão envolver e devem ser fechadas após o conteúdo envolvido, ou seja, elas possuem a tag de abertura e um correspondente de fechamento, que sempre será representado pela repetição do nome da tag, precedido de barra (/).
